Featured Project

LLM Prompt Engineering & Annotation (Child-Adult Dialogues)

This project explores how Large Language Models (Gemini 2.5 Flash-Lite) can annotate child-adult conversations according to cognitive linguistic guidelines. My role focuses on iteratively refining the annotation prompts and guidelines using real transcripts to improve the model's outputs.

This work represents the first stage of a larger pipeline. Later stages, including supervised fine-tuning (SFT) and additional model evaluation, will be carried out by the research team.

  • Analysed LLM outputs to identify and correct errors in discourse-level annotation.
  • Authored and refined annotation guidelines to steer model behaviour and resolve (potential) logical contradictions in earlier versions.
  • Created examples and rule-based instructions to guide the model for nuanced cases, e.g. distinguishing confirmation questions from information-seeking questions.
  • Refined schema definitions using Rumelhart & Norman's (1978) Modes of Learning Theory to improve the model’s identification of learning behaviours.
  • Refined Python-based prompt logic in Google Colab to fix factual inconsistencies, incomplete outputs, and over-tagging.
  • Added a human-scoring field to the required JSON output format to enable manual validation of LLM annotations.
