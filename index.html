Featured Project

LLM Prompt Engineering & Annotation (Child-Adult Dialogues)

This project documents a systematic, iterative approach to prompt engineering, aimed at benchmarking and enhancing an LLM’s ability to annotate child-adult interaction based on guidelines grounded in cognitive linguistics.

  • Systematically analysing LLM outputs to identify and correct errors in discourse-level annotation.
  • Authoring and refining comprehensive annotation guidelines to steer model behaviour, resolving logical contradictions found in previous versions.
  • Developing explicit, rule-based instructions to handle nuanced cases, such as ignoring non-meaningful conversational fillers.
  • Engineering prompts grounded in academic frameworks (eg. Rumelhart & Norman, 1978) to improve the model's conceptual understanding.
  • Designing JSON output structures to include evaluation metrics for ongoing performance tracking.
